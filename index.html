笔记

  1. HTML 链接由 标签定义。链接的地址在 href 属性中指定
  2. 标签在 HTML 页面中创建水平线。
  3. 如果您希望在不产生一个新段落的情况下进行换行（新行），请使用
    标签：

    This is
    a para
    graph with line breaks

  4. background-color 属性为元素定义了背景颜色；font-family、color 以及 font-size 属性分别定义元素中文本的字体系列、颜色和字体尺寸；text-align 属性规定了元素中文本的水平对齐方式
  5. HTML 元素定义短的引用。HTML
    元素定义被引用的节。浏览器通常会对
    元素进行缩进处理。
  6. HTML 元素定义缩写或首字母缩略语：

    WHO 成立于 1948 年。

  7. HTML
    元素定义文档或文章的联系信息（作者/拥有者）。HTML 元素定义著作的标题。
  8. HTML 元素定义双流向覆盖（bi-directional override）：This text will be written from right to left
  9. 通常，HTML 使用可变的字母尺寸，以及可变的字母间距。在显示计算机代码示例时，并不需要如此。, , 以及 元素全都支持固定的字母尺寸和间距。 元素不保留多余的空格和折行如需解决该问题，您必须在 
     元素中包围代码
  10. HTML 元素定义数学变量：

    E = m c2

  11. 超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，您可以点击这些内容来跳转到新的文档或者当前文档中的某个部分。有两种使用 标签的方式：1.通过使用 href 属性 - 创建指向另一个文档的链接2.通过使用 name 属性 - 创建文档内的书签
  12. name 属性规定锚（anchor）的名称。当使用命名锚（named anchors）时，我们可以创建直接跳至该命名锚（比如页面中某个小节）的链接，这样使用者就无需不停地滚动页面来寻找他们需要的信息了
  13. 首先，我们在 HTML 文档中对锚进行命名（创建一个书签）：基本的注意事项 - 有用的提示，然后，我们在同一个文档中创建指向该锚的链接：有用的提示
